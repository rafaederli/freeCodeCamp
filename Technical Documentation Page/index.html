Technical Documentation - Python

Introduction

Python is a high-level, interpreted programming language with dynamic typing. Created by Guido van Rossum and released in 1991, Python stands out for its simplicity and readability, making it one of the most popular and widely used languages in the programming world.

Features
  • Simple and Readable Syntax: Python is designed to be easy to read and write.
  • Cross-platform: Python runs on various operating systems, including Windows, Linux, and macOS.
  • Libraries and Frameworks: Python has a vast range of libraries and frameworks for web development, data science, automation, artificial intelligence, and more.
  • Community-driven: The Python community is active and collaborative, offering a wealth of tutorials, examples, and support.
Installation

How to install

  1. Windows:
    • Go to the official Python website python.org/downloads and download the Windows installer.
    • Run the installer and follow the instructions. Be sure to check the “Add Python to PATH” option during installation.
  2. MacOS:
    • Python is pre-installed on macOS, but you can install the latest version using Homebrew: 
                          
                      brew install python
  3. Linux:
    • Python is usually pre-installed on Linux. To check, type: 
                          
                      python3 --version
    • If it's not installed, use your package manager to install it:
      • Ubuntu/Debian: 
                                
                          sudo apt install python3
Basic Concepts

Variables and Data Types

In Python, you don't need to explicitly declare the variable type. The type is inferred automatically.

            
              x = 10            # Integer
              name = "Python"   # String
              pi = 3.14         # Float
              active = True     # Boolean

Conditional Statements

The if statement allows you to execute blocks of code based on conditions:

            
              x = 10
              if x > 5:
                  print("x is greater than 5")
              else:
                  print("x is less than or equal to 5")

Loops

The for loop allows you to iterate over a sequence of items, such as lists or strings:

            
              for i in range(5):
              print(i)

Functions

Functions in Python are defined using the def keyword:

            
              def greet(name):
                  return f"Hello, {name}!"

              print(greet("World"))
Working with Lists

Lists are ordered collections that can contain items of different types.

          
            fruits = ["apple", "banana", "orange"]
            print(fruits[0])  # Accessing the first item in the list
            fruits.append("grape")  # Adding an item
File Handling

Python allows you to easily open, read, and write files. Here's an example of reading a file:

          
            with open("example.txt", "r") as file:
            content = file.read()
            print(content)

Writing to Files

To write to files, use the w mode:

            
              with open("output.txt", "w") as file:
              file.write("This is an example file")
Popular Libraries

NumPy

NumPy is a fundamental library for scientific computing. It provides support for multidimensional arrays and mathematical functions.

            
              import numpy as np

              arr = np.array([1, 2, 3])
              print(arr + 1)  # Output: [2 3 4]

Pandas

Pandas is a powerful library for data analysis. It offers structures like DataFrame for working with tabular data.

            
              import pandas as pd

              data = pd.DataFrame({
                  "Name": ["Ana", "John", "Maria"],
                  "Age": [23, 34, 25]
              })
              print(data)

Flask

Flask is a micro-framework for web development.

            
              from flask import Flask

              app = Flask(__name__)

              @app.route("/")
              def hello_world():
                  return "Hello, World!"

              if __name__ == "__main__":
                  app.run()
Advanced Concepts

Object-Oriented Programming

Python supports object-oriented programming (OOP), allowing you to create classes and objects.

            
              class Car:
                  def __init__(self, model, color):
                      self.model = model
                      self.color = color

                  def start(self):
                      print(f"The {self.model} is started.")

              car = Car("Fusca", "blue")
              car.start()

Exception Handling

Exceptions can be handled using try and except to prevent the program from crashing.

            
              try:
                  result = 10 / 0
              except ZeroDivisionError:
                  print("Cannot divide by zero.")
Conclusion

Python is a versatile and powerful language that can be applied in a wide range of fields, from web development to data analysis and artificial intelligence. With its simple syntax and large community, Python is an excellent choice for both beginners and experienced developers.
